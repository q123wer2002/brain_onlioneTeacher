IP设置
允许进入整站後台的IP:
(留空则不限制,可以设置只允许哪些IP进入後台,每IP换一行,比如
192.168.0.2
192.168.1.5
192.168.0

如果某行设置成192.168.0 那么192.168.0.1-192.168.0.255这范围内的IP都是在允许的范围内
前的IP是: $onlineip
注意:如果的IP不是固定的话,比如是ADSL拨的话,不要填,否则IP更换後就进不了後台,此r只能通过修改後台文件强制进入後台进行修改恢复
)
允许访问前台W站的IP:
(留空则不限制,可以设置只允许哪些IP访问W站,每IP换一行,比如
192.168.0.2
192.168.1.5
192.168.0

如果某行设置成192.168.0 那么192.168.0.1-192.168.0.255这范围内的IP都是在允许的范围内
注意:这里只限制访问前台,但不限制登後台
)
封IP:
(可以设置封掉哪些IP访问W站前台和後台,每IP换一行,比如
192.168.0.2
192.168.1.5
192.168.0

如果某行设置成192.168.0 那么192.168.0.1-192.168.0.255这范围内的IP都是在受封的范围内
注意:这里会同r禁止访问前台与後台,不要封自己的IP,否则很麻烦,需要修改文件data/config.php查找到的IP把他删除才行
)
